# twarcUI
UI for twarc

This is a inital sketch for an UI for [twarc](https://github.com/DocNow/twarc) a Python library for archiving Twitter JSON data.

## RUNNING

    pip install pipenv

    pipenv install

    pipenv shell

    Edit config.py

    python db_create.py

    python worker.py (in a separate terminal window)

    python run.py
